# 안전을 위한 주의사항

:::danger

주의사항을 반드시 숙지하세요.

그렇지 않을 경우 상해가 발생하거나 로봇이 손상될 수 있습니다.

:::

### 배터리 잔량

배터리가 충분하지 않은 상태에서 로봇이 작동할 경우 갑작스럽게 로봇이 꺼질 수 있습니다.

### 비상정지

로봇이 예상치 못한 동작을 할 경우 즉시 비상정지 버튼을 눌러 로봇을 멈추세요.

### 다리 작동부 끼임 주의

로봇의 다리 작동부에 신체 부위가 끼이지 않도록 주의하세요.

### 충분한 공간 확보

로봇이 원활하게 작동할 수 있도록 주변에 충분한 공간을 확보하세요.
