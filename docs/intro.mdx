# 소개

<div style={{ textAlign: "center", paddingBottom: "20px" }}>
  <img src="/img/aidin8-ptz.png" alt="AIDIN" width="400" />
</div>

**AIDIN**은 복잡한 환경에서 최대 20kg의 다양한 탑재장비를 장착해 운용할 수 있는 4족 보행 로봇입니다.

발전소나 화학공장 등 사람이 근무하기 유해하거나 열악한 산업 현장에서 시설 점검과 유지 보수, 정찰을 수행할 수 있습니다.

## 특징

- 환경에 구애받지 않고 다양한 곳에 적용 가능
- 모듈화된 설계로 고객 요구에 따라 성능 조절이 가능하고 유지 보수가 간편
- 현장 환경에 맞추어 다양한 장비를 부착해 맞춤형 솔루션 제공 (실화상, 열화상 카메라, 감시, 소리, 진동, 가스 탐지 등)

## 사양

#### 물리적 특성

| 항목     | 사양         |
| -------- | ------------ |
| 크기     | 0 x 0 x 0 mm |
| 무게     | 60kg         |
| 보행속도 | 1.2m/s       |
| 방수등급 | IP67         |

#### 전원

| 항목        | 사양   |
| ----------- | ------ |
| 배터리 용량 | 0 Wh   |
| 구동시간    | 약 0분 |
| 충전방식    | 무선   |

#### 탑재 능력

| 항목           | 사양         |
| -------------- | ------------ |
| 최대 적재 크기 | 0 x 0 x 0 mm |
| 최대 적재 하중 | 20kg         |

#### 연결성

| 항목          | 사양           |
| ------------- | -------------- |
| Wi-Fi         | 2.4GHz 및 5GHz |
| 셀룰러 (선택) | 4G LTE         |

## 탑재장비

실화상 및 열화상 카메라, 가스 탐지기 등 현장에서의 임무를 수행하기 위한 장비입니다.
