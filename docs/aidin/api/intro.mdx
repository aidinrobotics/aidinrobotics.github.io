# API 개요

API를 통해 로봇을 제어하고 실시간 상태를 확인할 수 있습니다.

## 주요 기능

- **로봇 제어**: 로봇의 이동, 탑재장비 제어, 임무 수행 등을 원격으로 제어
- **실시간 상태 확인**: MQTT를 통한 상태 실시간 확인
- **임무 관리**: 임무 생성, 조회, 수정, 삭제

## API 유형

### REST API

HTTP 기반의 RESTful API로, 로봇 및 임무 관리를 위한 기본 CRUD 작업을 제공합니다.

- **기본 주소**

  `https://{robot}/api/v1`

- **형식**

  `Content-Type: application/json`

### MQTT API

로봇과의 실시간 통신을 위한 MQTT 프로토콜을 지원합니다.

- **Broker 주소**

  `mqtts://{broker}:8883`
